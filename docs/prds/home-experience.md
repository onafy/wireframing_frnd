# Home Experience PRD

## Summary
Homepage frndOS yang membantu user memahami nilai utama produk, menemukan module penting, dan masuk ke flow kerja yang paling relevan tanpa kebingungan.

## Requirements
- User bisa memahami value proposition frndOS dalam beberapa detik pertama
- User bisa melihat module utama yang tersedia dari homepage
- User bisa memahami sinyal, tools, dan recent work sebagai blok utama halaman
- User bisa masuk ke Studio dari CTA atau navigasi yang jelas

## Notes
Wireframe awal berfokus pada desktop experience dan clarity of hierarchy sebelum masuk ke eksplorasi responsive detail.
